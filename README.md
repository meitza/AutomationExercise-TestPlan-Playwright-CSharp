# AutomationExercise-TestPlan-Playwright-CSharp
Playwright + C# automation testing project for AutomationExercise.com created for the Quest Global QA Internship Test. Includes a custom test plan, automated UI test scenarios, cart validation, navigation testing, responsive/mobile testing, and additional edge-case scenarios using NUnit and Playwright.

TestCase1.cs to TestCase26.cs are the original example test cases provided by the website https://automationexercise.com/in the “Test Cases” section.

MyTestCase1.cs to MyTestCase10.cs are additional automated test scenarios created by me for the website https://automationexercise.com

TestPlan.pdf contains the custom test plan created for the additional automated tests, including testing objectives, test environment, testing scenarios, expected results and application improvement suggestions.
